# okaselect
wip

## usage

```sh
yarn add okaselect
```

```js
import * as okaselect from 'okaselect'
```

## commnad

```sh
# install dependencies
$ yarn install

# lint
$ yarn lint

# test
$ yarn test [--watch]

# build
$ yarn build

# generate doc
$ yarn doc

# serve demo at localhost:1234
$ yarn demo
```

## publish
Update `version` in `package.json`, commit with a comment `Release x.x.x` and merge into the `main` branch.
